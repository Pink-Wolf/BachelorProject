Graphics.Core

Contains fundamental logic for creating windows, rendering, and doing other gpu-related work.

Header-files:

  • basic_drawables: Contains some simple resources that can be used to draw things on a window.
  • drawables: Contains basic logic for drawing objects on a window with drawable.
  • gpus: Contains general logic for gpus and their threads.
  • gpu_buffers: Contains various buffers for data used by shaders.
  • gpu_programs: Contains gpu_program to actually run code on the gpu.
  • graphics: Contains graphics_environment, used to set up and manage program-wide logic used for connecting to the gpus.
  • inputs: Contains general logic for handling inputs using input_manager.
  • shaders: Contains general logic for gpu-code.
  • windows: Contains logic for creating and managing windows.
