CompWolf::window

In project Graphics.Core

Defined in header <windows>

Member of

Inherits from:

class window;

A window, as in a rectangle that can display a dynamic image, and that listens for various inputs from outside the program (like a key being pressed).

Warning

Template Parameters

Members

NameDescription
(constructors)window is not copyable, but is movable.

Accessors

gpuReturns the gpu that the window is on.
inputsReturns the input_manager handling inputs while the window is focused.
pixel_sizeReturns the size of the window, in pixels.

Modifiers

Vulkan-related

These members reveal a lower level of abstractions, and so they should rarely be used.

surfaceReturns the surface of the window.

rebuilding_surfaceEvent invoked before the window's surface is rebuild.
rebuild_surfaceEvent invoked after the window's surface is rebuild.

Example

The following code creates an environment, uses it to create a window, and then keeps them alive until the window is closed.

#include <graphics> // graphics_environment, graphics_environment_settings
#include <windows> // window, window_settings
using namespace CompWolf;

int main()
{
	graphics_environment environment(graphics_environment_settings{
		.program_name = "Test Program",
		});

	window win(environment, window_settings{
		.name = "Test Window",
		});

	while (win.running())
	{
		environment.update();
	}
}
