Your First Window

This tutorial will go over how to create a window using the library CompWolf.Graphics.Core.

graphics_environment

Most features of CompWolf.Graphics requires a graphics_environment to work. The graphics_environment is defined in the graphics-header. Creating an environment requires a graphics_environment_settings, which can be used to adjust how the environment functions. For this tutorial, we can just let it function as normal. To create a program that creates a graphics_environment, write the following code:
#include <graphics>

CompWolf::graphics_environment environment;

int main()
{
    CompWolf::graphics_environment_settings environment_settings;

    environment = CompWolf::graphics_environment(environment_settings);
}

window

With a graphics_environment, you can now create a window. The window is defined in the windows-header. Like the graphics_environment, creating a window requires a window_settings, which can be used to adjust how the window functions. Creating a window also requires a graphics_environment. Write the following code to create a window:
#include <windows>
CompWolf::window_settings win_settings{
    .name = "Hello World",
};
CompWolf::window win(environment, win_settings);
If we run the code now, the window will close immediately. window has a method running(), which tells us if the window has not been closed. We can therefore wait until someone closes the window with a while loop:
while (win.running()) {}
The window will now stay open, but we cannot close it. We need to continually call environment.update() to be able to interact with the window. Therefore, to make the window stay open and be interactable, write:
while (win.running())
{
    environment.update();
}
Congratulations, you have now created a window.

Final code

#include <graphics>
#include <windows>

CompWolf::graphics_environment environment;

int main()
{
    CompWolf::graphics_environment_settings environment_settings;

    environment = CompWolf::graphics_environment(environment_settings);

    CompWolf::window_settings win_settings{
        .name = "Hello World",
    };
    CompWolf::window win(environment, win_settings);

    while (win.running())
    {
        environment.update();
    }
}
